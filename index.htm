Evolutionary history of Euphorbia polycarpa
Evolutionary study of the complex from Sonoran Desert. It has multiple morphological variation and for that some authors designated them to varieties. DNA were used to clarify the relation between these varieties.

Novelities of Euphorbia from Baja California Peninsula
New species of Euphorbia were described (Euphorbia vizcainensis and E. cerralvensis). They occurring in the Peninsula of Baja California. Confirmed as separate entities using DNA from others relatives.
More...
Review of Dalechampia from Colombia
The revision of Dalechampia is a subject very interesting for me. I had been worked on that for sometime. I am looking for someone that want to continue in collaborative team using all my information.
More...
Vines Euphorbiaceae of São Paulo state
A study of the taxonomy of the vines genus of Euphorbiaceae from São Paulo stated was done. 17 species in 5 genus ( Bia, Dalechampia, Plukenetia, Romanoa and Tragia) were reported.
More...
Euphorbiaceae from Quindío (Colombia)
A complete revision of all Euphorbiaceae that occurring in Quindío state was done. It include bibliographical review, descriptions, and identification keys for all species.
More...
ChecklistBuilder
ChecklistBuilder is a printer report for Windows, specialliced in Botanical Checklist and other diversity list. Can be easily customized the reports and used a hierarchical structure for synonyms and other information.
More...
ColectoR
ColectoR is an assistant for botanist field trips, to help to manage big amounts of data from each specimen collected using any Android smarthphone based. It presents an interface to make easy the exportation of data to use directly in Excel.
More...
Label printer for ColectoR
This is a spreadsheet in Excel that help you to print labels in a easy and quick way. This file can be used as companion of ColectoR due their integration with the exported format of the database. It can be used as main database for collectors.
More...
SinóptiK
This software for Windows is a interactive synoptic key for the Euphorbiaceae species that occurring in the Quindío state (Colombia). Also it includes different package to create multiples databases from other groups.
