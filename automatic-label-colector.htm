Label printer for ColectoR

C.A. Maya-Lastra - 2014

Avalible for: Windows or Mac (Excel is required)

As a complement of ColectoR, I created and automated spreadsheet for Excel, where the database built for ColectoR can be imported to print the labels for the specimens registered.

The automatic label printer has three sheets:
  1. Presentation sheet
  2. Main database, here can be paste all information from the exported database obtain from ColectoR. You must notice that in some languages some characters should not been displayed correctly, use the information in ColectoR to manage the exported database.
  3. Print sheets, here you can write the number of collection that you want to print. To forty labels can be printed in one process.
