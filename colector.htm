ColectoR

C.A. Maya-Lastra - 2014-2019

Avalible for: Android

ColectoR is a free assistant for botanist field trips, to help to manage big amounts of data from each specimen collected using any Android smarthphone based. ColectoR contain the fields most used by botanist in the label for their collections. ColectoR use sensors in each smartphone like GPS, altimeter and camera, allowing integrate this information with each registry. Present functions to avoid enter repetitive data like main collector, additional collector, country and others. Auto-increment the number of collector and suggest morphological descriptions based on the previews registries entered. Use a API of inverse geocoding to suggest names of locations, roads and others. ColectoR can connect directly with iPlant CollaborativeTM servers to verify species names, authors and families. ColectoR present an interface to make easy the exportation of data to use directly in Excel or other similar. In ColectoR you can find rapid entry lists to enter data that you used in other past registries.

ColectoR was thought-out to be easy, fast and productive.

Field trips of 1 to 4 weeks driving long distances, can be result in hundred of specimens with a big notes and labels unsorted. The elaboration of labels can be result in many hours of work registering each collection in the main database. ColectoR permit finish the trip with the database almost done ready to export to create labels or reports in minutes.

    Secure your data:
    Remember ColectoR save every change in a backup file (in /ColectoR_Data/ColectoR_backup.txt) this is very useful, in the follow cases to rescue some data: uninstall the app; bad execution or app crashes;out of memory errors. If you save a entry, modify, or even delete, this will be permanently in the backup file. You can import in excel and select the desired information.
    The new add in this version is the camera fuction, remember this is a Beta function, can be very well, but their execution depends in a good way of each smartphone. If your app crashes or close and you cannot save the photos, in the folder /ColectoR_Photos/TEMP/ are the last unsaved session photos.

List of features:
• Minimalist, intuitive and graphic interface
• Auto-fill of repetitive data as country, collector name and additional collector
• Auto-increment of collector number
• Fast data entry with helpers and rapid entry lists
• Verification of scientific names in iPlant CollaborativeTM through cellular network

Complete list of features (in Spanish)
- Interfaz minimalista, intuitiva y gráfica
- Edición y corrección de cada registro con un solo botón
- Contiene los campos más usados en las etiquetas botánicas
- Rápida entrada por registro reduciendo el tiempo de captura de datos en campo
- Registro fotográfico de soporte
- Múltiples posibilidades de respaldo en línea de la base de datos por medio de una interfaz para exportar a sistemas como Dropbox, Google Drive, OneDrive, Box, Mega, Gmail, entre otros
- Autollenado de datos repetitivos como país, colector y colectores asociados
- Autoincremento del número de colector
- Listados de ingreso rápido de entradas y datos usados en otros registros
- Verificación y corrección de nombres de especies por medio de los servidores de iPlant CollaborativeTM
- Obtención de datos de los sensores del celular como altímetro y GPS
- Sugerencias de localidad usando geocodificación inversa de Google
- Accesos directos para reconocimiento de voz para llenar campos largos como la descripción de la planta y notas
- Sugerencias de descripciones de registros anteriores, que permite unificar la descripción de organismos similares sin perder datos importantes
- Integración con objetos nativos de Android para ofrecer servicios como corrección ortográfica y acceso al portapapeles
- Base de datos en formato universal (*.csv) para permitir la importación sencilla desde procesadores de hojas de cálculo (como Excel) y de bases de datos
- Tamaño reducido de la base de datos y uso de APIs minimalistas para reducir gastos en redes celulares
- Adaptación a cualquier tamaño de pantalla, desde pequeños celulares inteligentes hasta tabletas de gran tamaño
- Grabado de datos en la memoria externa (en caso de tenerla) para mejorar la movilidad y resguardo de los datos
